Verb Registries

The following [ages contain lists of verbs which you should use. Please do not define your own verbs if you could use one of the existing verbs on these pages.

  • ADL verbs and activities list.
  • Activity Streams verb registry (Read Megan's article first!)
  • This page! moderated by Andrew Downes (@mrdownes) of tincanapi.co.uk
  • verb registry maintained by Russell Duhon (@fugo13) of Saltbox Services
  • vocabulary registry (list of lists of verbs) maintained by Ingo Dahn of University Koblenz-Landau

Verb List

This section lists all verbs defined at tincanapi.co.uk.

Note that during beta, different URIs were used. These use the same verb names but began with the base URI http://www.tincanapi.co.uk/wiki/verbs:. Activity providers should not use these old ids, even if targeting a beta version of the specification.

Reporting tools supporting the 0.95 version of the specification may wish to treat http://www.tincanapi.co.uk/wiki/verbs: as equivelent to 'http://www.tincanapi.co.uk/verbs/' when found as part of a verb id URI. There is no need for reporting tools to support this deprecated form in 1.0 and later versions of the specification.

started

id: http://adlnet.gov/expapi/verbs/initialized

deprecated id: http://www.tincanapi.co.uk/verbs/started

  • Marks the beginning of a session
  • Equivalent to SCORM's "Initialize"
  • More info: Guide:SCORM_to_Tin_Can_API/Part_3_-_Summary_of_When_to_Send_Tin_Can_API_Statements_0_95

stopped

id: http://adlnet.gov/expapi/verbs/terminated

deprecated id: http://www.tincanapi.co.uk/verbs/stopped

  • Marks the end of a session
  • Equivalent to SCORM's "Terminate"
  • More info: Guide:SCORM_to_Tin_Can_API/Part_3_-_Summary_of_When_to_Send_Tin_Can_API_Statements_0_95

evaluated

id: http://www.tincanapi.co.uk/verbs/evaluated

  • Used for evaluating a learning activity.
  • Object should be a "StatementRef".
  • More info: Guide:Tin_Can_Learning_Evaluator#Evaluating_and_Reflecting

enrolled onto learning plan

id: http://www.tincanapi.co.uk/verbs/enrolled_onto_learning_plan

  • Part of the Learning Plan Framework
  • Object should be an "Activity".
  • More info: The_Learning_Plan_Framework#Enrolled

planned

id: http://www.tincanapi.co.uk/verbs/planned_learning

  • Part of the Learning Plan Framework
  • Object should be a "subStatement".
  • More info: The_Learning_Plan_Framework#Planned

cancelled planned learning

id: http://www.tincanapi.co.uk/verbs/cancelled_planned_learning

  • Part of the Learning Plan Framework
  • Object should be a "StatementRef".
  • More info: The_Learning_Plan_Framework.html#Cancelled
